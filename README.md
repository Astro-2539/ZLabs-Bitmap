# Z Labs Bitmap
「Z Labs Bitmap」是一款仿 [GNU Unifont](https://www.unifoundry.com/unifont/index.html) 风格的小型像素字体。这套字体在 11 * 12 的像素空间（实际占用 12 * 12 像素）中，尽可能还原了 Unifont 字体的特色。

本字体遵循中国大陆标准规范字形，西文字体按等宽规格设计。

> **注意：** 此字体目前处于早期开发阶段，尚有大量字符没有制作（详见「覆盖范围」一节），敬请谅解。

> **提示：** 本字体为开源项目，采用 OFL 许可证授权，您可以免费商用此字体（详见「授权」一节）。

## 关于「等宽版本」
本字体具有普通版本（字体名称为"Z Labs Bitmap"）和等宽版本（字体名称为"Z Labs Bitmap Mono"）两种版本。

由于字体设计时即为按照等宽规格进行设计，因此两个版本的字形完全一致，区别仅在于字体属性中是否有「支持等宽」属性（类似于「宋体」与「新宋体」的区别）。

制作等宽版本的目的是使得本字体能够在仅支持具有**支持等宽属性**字体的软件和游戏中被套用。

出于兼容性等原因，本字体没有直接为原始版本字体添加「支持等宽」属性，而是单独制作了具有「支持等宽」属性的变体字体。二者将同步更新。
