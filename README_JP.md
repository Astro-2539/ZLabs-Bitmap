# Z Labs Bitmap 12px

「Z Labs Bitmap 12px」は、サイズが 12px のピクセルフォントです。日本語、中国本土、香港の 3 種類の文字形状をサポートしており、西文文字は等幅仕様で設計されています。
 
> [!WARNING]
>
> このフォントは現在、字形調整段階にあります。このフォントはすでに日本語、簡体字中国語、繁体字中国語の必要文字を完全にサポートしていますが、一部の文字についてはまだ調整が必要です。現在のリリース候補版は最終版の品質を保証するものではありません。
> 
> 使用中に字形の誤りや不調和などの問題を見つけた場合は、速やかに「Issues」でご報告ください。
>

> [!IMPORTANT]
>
> これはオープンソースプロジェクトであり、フォント部分は [OFL-1.1](https://openfontlicense.org/open-font-license-official-text/) ライセンスに基づいています。このフォントを使用する際、著者に別途ライセンスを購入する必要はありません（商業利用やアプリ／ハードウェアへの組み込みなども含む）。

## 字形のバリエーション
 
> [!IMPORTANT]
>
> プロジェクトをローカライズする際、適切なフォントを選ぶことが非常に重要です。簡体字中国語には CN バージョン、繁体字中国語には HC バージョン、日本語には JP バージョンを推奨します。

> 三者の西文部分が同一であるため、英語やフランス語などの言語のローカライズにはいずれのバリエーションを使用しても構いません（ただし、一部の記号は中国語の使用習慣に従って描画します）。

#### フォントの戻し
 
著者の精力には限りがあるため、現在のHC版とJP版では一部の漢字のみが作成されています。そのため、使用中に欠落した漢字に遭遇する可能性があります。
 
ソフトウェアがフォントのロールバック機能をサポートしている場合、`font_family` または類似の属性を設定することで、漢字数が多い CN バリアントを HC/JP のバックアップフォントとして指定し、CN バリアントで HC/JP がまだ対応していない漢字を表示できるようにすることができます。 
 
ソフトウェアがこの仕組みをサポートしていない場合、対応するバリエーションの「フォールバック」（FB）バージョンをご検討ください。このバージョンでは、HC/JP バリエーションで既にサポートされている漢字はそのまま維持され、CN バリエーションで作成済みで HC/JP では未作成の漢字については、CN の文字形状を対応するコード位置に埋め込みます（その文字が HC/JP の作成計画に含まれていようが含まれていまいが関係ありません）。これにより、漢字をサポートする文字数は CN 版と一致しますが（文字形状の一致は保証されません）、文字形状の整合性は確保されません。
 
 
## エンジニアリングファイルからフォントを構築する
 
本フォントは [Bits'n'Picas](https://github.com/kreativekorp/bitsnpicas) を用いて作成されました。`./tools/build.py` を実行すると、フォントを生成できます。
 
ビルドプロセスには、`fonttools` ライブラリ、`pixel_font_builder` ライブラリ、および `kbitfont` ライブラリが依存しています。 
構築プロセスの詳細は、`Tools` フォルダー内のリードミーをご覧ください。
